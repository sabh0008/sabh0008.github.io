About Bootstrap!

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components. Bootstrap is easy to use, user friendly framework which help students and makes the work easier for students who are new at coding and dont know much about framework and styling.

Till now there have been many development in bootstrap and its software as we have 4 bootstrap versions on the table, but for now BootStrap-4 is being used as it is the most updated version with more advanced plugins and frameworks.
